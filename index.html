To begin with, when describing a sample space, we want a structure where given two events, \(\displaystyle A\) and \(\displaystyle B\), we can say that the following are also events:

  • \(\displaystyle A\) AND \(\displaystyle B\), where both events occur,
  • \(\displaystyle A\) OR \(\displaystyle B\), where one or both events may occur,
  • NOT \(\displaystyle A\), where \(\displaystyle A\) does not occur (similarly for \(\displaystyle B\))
  • NOTHING, where nothing happens at all.

The tools of set theory give us a means to this end. If events are said to be sets, then the structure we want is that of a \(\displaystyle \mathbf{\sigma }\)-algebra: given a set \(\displaystyle S\) a \(\displaystyle \mathbf{\sigma }\)-algebra over \(\displaystyle S\) is a set \(\displaystyle \mathfrak{S}\) of subsets of \(\displaystyle U\) which satisfy the following:

  • \(\displaystyle \emptyset \in \mathfrak{S}\) (NOTHING is an event)
  • if \(\displaystyle A\in \mathfrak{S}\), then \(\displaystyle A^{c} \in \mathfrak{S}\) (NOT \(\displaystyle A\) is an event)
  • if \(\displaystyle A_{i} \in \mathfrak{S}\) for all \(\displaystyle i\in \mathbb{N}\), \(\displaystyle {\displaystyle \cup }{\textstyle _{i} A_{i} \in }\mathfrak{S}\) (\(\displaystyle A_{1}\) OR \(\displaystyle A_{2}\) OR \(\displaystyle A_{3}\) OR... is an event)

The fact that the last property can apply to countably many sets \(\displaystyle A_{i}\) is crucial. For instance, the set of finite subsets of \(\displaystyle \mathbb{N}\) as well as their complements is not a \(\displaystyle \sigma \)-algebra because there are infinite subsets of \(\displaystyle \mathbb{N}\) with infinite complements (for example, the postive even numbers).


We can talk about the \(\displaystyle \sigma \)-algebra generated by a collection \(\displaystyle A\) of subsets of \(\displaystyle S\), denoted as \(\displaystyle \sigma ( A)\), as being the intersection of all \(\displaystyle \sigma \)-algebras containing \(\displaystyle A\), which is also a \(\displaystyle \sigma \)-algebra. The Borel sets \(\displaystyle \mathfrak{R}^{d} =B\left(\mathbb{R}^{d}\right)\) are defined as being the \(\displaystyle \sigma \)-algebra generated by the open sets in \(\displaystyle \mathbb{R}^{d}\).

The pair \(\displaystyle ( S,\ \mathfrak{S})\) is called a measurable space. A measurable space should contain sets to which a measure, i.e. a notion of "size" or "volume", can be assigned. The \(\displaystyle \sigma \)-algebra ensures that the sets are closed under operations that would be reasonable for such measurable sets - the complement of a measurable set is also measurable, and the countable union of measurable sets is also measurable. A measure is defined precisely as a function \(\displaystyle \mu :\mathfrak{S}\rightarrow \mathbb{R}\) that satisfies the following:

  • \(\displaystyle \mu ( A) \geq 0\) for all \(\displaystyle A\in \mathfrak{S}\) (measures are positive)
  • \(\displaystyle \mu ( \emptyset ) =0\) (empty set has measure zero)
  • \(\displaystyle \mu \left(\bigcup _{i} A_{i}\right) =\sum _{i} \mu ( A_{i})\) for all countable sequences \(\displaystyle \{A_{i}\}_{i\in \mathbb{N}}\) of disjoint sets in \(\displaystyle \mathfrak{S}\)

The last property ensures that the "sizes" of disjoint sets are "added" together.

If \(\displaystyle \mu ( S) =1\) we say that \(\displaystyle \mu \) is a probability measure. Then if a set \(\displaystyle A\in \mathfrak{S}\) corresponds to an event, \(\displaystyle \mu ( A)\) would then correspond to the probability of that event occuring. A measure space, together with a probability measure, is refered to as a probability space.

Some measures are common:

  • Over a finite set \(\displaystyle S\), the measure \(\displaystyle \mu ( \omega ) =\frac{1}{|S|}\) for all \(\displaystyle \omega \in S\) (the discrete uniform distribution)
  • Over \(\displaystyle (\mathbb{R} ,\ \mathfrak{R})\), the measure \(\displaystyle \mu (( a,b]) =b-a\)
    • This is called the Lebesgue measure.
    • When restricted to the interval \(\displaystyle ( A,B]\) for \(\displaystyle A,B\in \mathbb{R}\), \(\displaystyle \frac{\mu }{B-A}\) represents the continuous uniform distribution from \(\displaystyle A\) to \(\displaystyle B\).

These probability measures seem to correspond to some probability distribution. Since a probability distribution determines a random variable, it makes sense to ask how to rigourously define one.


First, given two measurable spaces \(\displaystyle ( S,\ \mathfrak{S})\) and \(\displaystyle ( T,\ \mathfrak{T})\), a measurable function is a function \(\displaystyle f:S\rightarrow T\) such that given \(\displaystyle B\in \mathfrak{T}\) the pre-image \(\displaystyle A=f^{-1}[ B]\) is in \(\displaystyle \mathfrak{S}\), i.e. the pre-image of a measurable set is also measurable. In other words, it "preserves" the structure of a measurable space.

A random variable is a function from a probability space \(\displaystyle ( S,\mathfrak{S} ,\ P)\) to \(\displaystyle (\mathbb{R} ,\ \mathfrak{R})\). An example of this is the indicator function of a set \(\displaystyle A\in \mathfrak{S}\):

\begin{equation*} 1_{A}( \omega ) := \begin{cases} 1 & \omega \in A\\ 0 & \omega \notin A \end{cases} \end{equation*}

A random variable \(\displaystyle X\) induces a probability distribution \(\displaystyle \mu _{X}( A) =P( X\in A) =P\left( X^{-1}[ A]\right)\). We can look at the specific sets \(\displaystyle A=( -\infty ,\ x]\) and define \(\displaystyle F( x) =\mu _{X}( -\infty ,x] =P( X\leq x)\) to be the cumulative distribution function of \(\displaystyle X\).


We can use measures to define Lebesgue integrals, which are essential in defining ubiquitous aspects of random variables such as expected value.

First and foremost, we can define integrals for a particular class of functions called simple functions, which are functions that can be written in the form \(\displaystyle \sum _{i=1}^{n} a_{i} 1_{A_{i}}\) for some \(\displaystyle n\in \mathbb{N}\), where \(\displaystyle a_{i} \in \mathbb{R} ,\ \{A_{i}\}_{i\in \mathbb{N}}\) is pairwise disjoint, and \(\displaystyle \mu ( A_{i}) < \infty \) for all \(\displaystyle i\). These functions are similar to step functions, with the caveat that step functions force the \(\displaystyle A_{i}\) to be intervals. For simple functions this need not be the case: for instance, one could use the rational numbers.

The integral of a simple function \(\displaystyle \varphi \) with respect to a probability measure \(\displaystyle \mu \) is then:

\begin{equation*} \int \varphi \ d\mu =\int \sum _{i=1}^{n} a_{i} 1_{A_{i}} \ d\mu := \sum _{i=1}^{n} a_{i} \mu ( A_{i}) \end{equation*}

For a bounded measurable function \(\displaystyle f\leq M\in \mathbb{R}\) with finite support \(\displaystyle E\) such that \(\displaystyle \mu ( E) < \infty \), the integral of \(\displaystyle f\) is: \begin{equation*} \int f\ d\mu := \sup _{\varphi \leq f}\int \varphi \ d\mu =\inf_{\psi \geq f}\int \psi \ d\mu \end{equation*}

where \(\displaystyle \varphi ,\ \psi \) are both simple functions and the supremum/infimum are both over the sets of simple functions less than/greater than \(\displaystyle f\). In a sense, the simple functions converge to the integral from above and below.

For an general non-negative measurable function \(\displaystyle f\geq 0\) the integral is the same but with the caveat that the simple function be non-negative over a set of finite measure.

Now we can define a general measurable function to be Lebesgue integrable if \(\displaystyle \int |f|\ d\mu < \infty \). The integral of such a function is defined as the difference between the positive and negative parts of the integral. If \(\displaystyle f_{+} =\max( 0,\ f)\) and \(\displaystyle f_{-} =\min( 0,\ f)\) are the positive and negative parts of \(\displaystyle f\) respectively, so that \(\displaystyle f=f_{+} -f_{-}\) then:

\begin{equation*} \int f\ d\mu := \int ( f_{+}) \ d\mu \ -\ \int f_{-}) \ d\mu \end{equation*}

And now, the Lebesgue integral has been defined. When \(\displaystyle ( S,\ \mathfrak{S} ,\ \mu ) =\left(\mathbb{R}^{n} ,\ \mathfrak{R}^{n} ,\ \mu \right)\) we write \(\displaystyle \int f( x) \ dx\), which coincides with the Riemann integral wherever the Riemann integral exists.

For integrating only over a subset of the domain \(\displaystyle A\subset S\), we write \(\displaystyle \int _{A} f\ d\mu =\int f\cdot 1_{A} \ d\mu \).


The Lebesgue integral has several key properties. Suppose, for the following, that \(\displaystyle f\) and \(\displaystyle g\) are integrable functions:

  1. If \(\displaystyle f\geq 0\) almost everywhere (a.e.), then \(\displaystyle \int f\ d\mu \geq 0\)
  2. For all \(\displaystyle a\in \mathbb{R}\), \(\displaystyle \int af\ d\mu =a\int f\ d\mu \)
  3. \(\displaystyle \int f+g\ d\mu \ =\ \int f\ d\mu \ +\ \int g\ d\mu \)
  4. If \(\displaystyle g\leq f\) a.e. then \(\displaystyle \int g\ d\mu \leq \int f\ d\mu \)
  5. If \(\displaystyle g=f\) a.e. then \(\displaystyle \int g\ d\mu =\int f\ d\mu \)
  6. \(\displaystyle \Bigl|\int f\ d\mu \Bigl| \ \leq \ \int |f|\ d\mu \)

In addition, there are specific results developed from these properties:

  • Jensen's inequality: Suppose \(\varphi \) is convex, that is,

\begin{equation*} \lambda \varphi ( x) +( 1-\lambda ) \varphi ( y) \geq \varphi ( \lambda x+( 1-\lambda ) y) \end{equation*}

for all \(\lambda \in ( 0,1)\) and \(x,y\in \mathbb{R} .\) The left hand side is a linear interpolation between the values \(\displaystyle \varphi ( x)\) and \(\displaystyle \varphi ( y)\), whereas the right hand side is the actual value of the function below that point. Then the following inequality holds: \begin{equation*} \int \varphi ( f) \ d\mu \geq \varphi \left(\int f\ d\mu \right) \ \end{equation*} For another inequality, we define the \(\displaystyle p\)-norm, for \(\displaystyle 1\leq p< \infty \): \begin{equation*} \| f\| _{p} =\left(\int |f|^{p} \ d\mu \right)^{1/p} \end{equation*} Functions for which this norm is finite form a Banach space called \(\displaystyle L^{p}( S,\ \mu )\), or simply \(\displaystyle L^{p}\).


H√∂lder's inequality: If \(\displaystyle p,q\in ( 1,\infty )\) where \(\displaystyle 1/p+1/q=1\) then: \begin{equation*} \int |fg|\ d\mu \ \leq \ \| f\| _{p} \cdotp \| g\| _{q} \end{equation*} What this tells us is that if \(\displaystyle f\in L^{p}\) and \(\displaystyle g\in L^{q}\) then the norms are finite and their product \(\displaystyle fg\in L^{1}\). In addition, if the integral on the left is infinite then one or both of the norms must be infinite as well. The specific case where \(\displaystyle p=q=2\) is called the Cauchy-Schwarz inequality.

We say that a sequence of measurable functions \(\displaystyle \{f_{n}\}_{n\in \mathbb{N}}\) converges in measure to a measurable function \(\displaystyle f\) if for every \(\displaystyle \epsilon >0\), \(\displaystyle \mu (\{x\ :\ |f_{n}( x) -f( x) | >\epsilon \})\rightarrow 0\) as \(\displaystyle n\rightarrow \infty \). Then if \(\displaystyle f\) vanishes on the complement of some finite-measure set \(\displaystyle E\), and \(\displaystyle |f_{n} |\) is bounded above by a constant \(\displaystyle M\), the bounded convergence theorem states that

\begin{equation*} \int f\ d\mu \ =\ \lim _{n\rightarrow \infty }\int f_{n} \ d\mu \end{equation*}

In a sense, since \(\displaystyle f_{n}\) converges to \(\displaystyle f\), we are able to "interchange the limits" here. Fatou's lemma states that for a general sequence \(\displaystyle f_{n} \geq 0\): \begin{equation*} \liminf _{n\rightarrow \infty }\int f_{n} \ d\mu \geq \int \left(\liminf _{n\rightarrow \infty } \ f_{n}\right) \ d\mu \end{equation*} An example of where there is inequality is for the sequence

\begin{equation*} \ f_{n} =\begin{cases} n & x\in ( 0,\ 1/n)\\ 0 & \text{otherwise} \end{cases} \end{equation*} On \(\displaystyle ( 0,1)\) equipped with Borel sets and the Lebesgue measure,\(\displaystyle \int f_{n} \ d\mu =1\) for all \(\displaystyle n\) but the functions converge pointwise to the zero function, with integral \(\displaystyle 0\).


The monotone convergence theorem follows from Fatou's lemma. If \(\displaystyle f_{n} \geq 0\) and \(\displaystyle f_{n}\) increases to \(\displaystyle f\), then we have that \(\displaystyle \int f_{n} \ d\mu \) also increases to \(\displaystyle \int f\ d\mu \). Fatou's lemma and the fact that \(\displaystyle f_{n} \leq f\) force the limits infimum and supremum to converge.

The dominated convergence theorem states that if \(\displaystyle f_{n}\rightarrow f\) almost everywhere and \(\displaystyle |f_{n} |\) is bounded above by an integrable function for all \(\displaystyle n\), then \begin{equation*} \int f_{n} \ d\mu \ \rightarrow \ \int f\ d\mu . \end{equation*} This follows from Fatou's lemma and the monotone convergence theorem.


These integral theorems can be used in places where a quantity may be defined via an integral. We define the expected value of a random variable \(\displaystyle X\) on a probability space \(\displaystyle ( S,\ \mathfrak{S} ,\ \mu )\) to be \begin{equation*} \mathbf{E}[ X] := \int X\ d\mu \end{equation*}

There are several key properties that follow from the properties of integrals:

Suppose \(X,Y\geq 0\) and \(\displaystyle \mathbf{E}[ X] ,\mathbf{E}[ Y] < \infty .\)

  • \(\mathbf{E}[ X+Y] =\mathbf{E}[ X] +\mathbf{E}[ Y] .\)
  • \(\mathbf{E}[ aX+b] =a\mathbf{E}[ X] +b\) for \(\displaystyle a,b\in \mathbb{R}\)
  • If \(X\geq Y\), then \(\mathbf{E}[ X] \geq \mathbf{E}[ Y]\)

Jensen's inequality. \begin{equation*} \mathbf{E}[ \varphi ( X)] \geq \varphi (\mathbf{E}[ X]) \end{equation*}

This follows from the general property that for convex \(\displaystyle \varphi \).

H√∂lder's inequality. If \(p,q\in [ 1,\infty ]\) with \(1/p+1/q=1\), then

\begin{equation*} \mathbf{E}| XY| \leq \| X\| _{p} \| Y\| _{q} \end{equation*} For the specific case when \(\displaystyle p=q=2\), the 2-norms are equivalent to the square root of the second moment, so that the following inequality holds: \begin{equation*} (\mathbf{E}[ XY])^{2} \leq \mathbf{E}\left[ X^{2}\right]\mathbf{E}\left[ Y^{2}\right] \end{equation*} We can rewrite this as the variance inequality by first noting the definition of covariance: \begin{equation*} \begin{aligned} \mathbf{Cov}( X,Y) & =\mathbf{E}[( X-\mathbf{E}[ X])( Y-\mathbf{E}[ Y])]\\ (\mathbf{Cov}( X,Y))^{2} & =(\mathbf{E}[( X-\mathbf{E}[ X])( Y-\mathbf{E}[ Y])])^{2}\\ & \leq \mathbf{E}\left[( X-\mathbf{E}[ X])^{2}\right] \cdot \mathbf{E}\left[( Y-\mathbf{E}[ Y])^{2}\right]\\ & =\mathbf{Var}[ X] \cdot \mathbf{Var}[ Y] \end{aligned} \end{equation*} Finally: \begin{equation*} \mathbf{Cov}( X,Y) \leq \sigma _{X} \sigma _{Y} \end{equation*}

Chebyshev's (or Markov's) inequality. Suppose \(\varphi :\mathbb{R}\rightarrow \mathbb{R}\) has \(\varphi \geq 0\), let \(A\in \mathit{R}\) and let \(i_{A} =\inf \{\varphi ( y) :y\in A\}\). Then: \begin{equation*} i_{A} P( X\in A) \leq E( \varphi ( X) ;X\in A) \leq E\varphi ( X) \end{equation*}

From the results where we switched the order of limits and integration, here are several other theorems:

Fatou's lemma. If \(X_{n} \geq 0\) then \begin{equation*} \liminf _{n\rightarrow \infty } \ E[ X_{n}] \geq \mathbf{E}\left(\liminf _{n\rightarrow \infty } \ X_{n}\right) \end{equation*}

Monotone convergence theorem. If \(0\leq X_{n} \uparrow X\) then \(\mathbf{E}[ X_{n}] \uparrow \mathbf{E}[ X] .\)

If \(X_{n}\rightarrow X\ a.s.,| X_{n}| \leq Y\) for all \(\displaystyle n\), and \(\mathbf{E}[ Y] \leq \infty \), then \begin{equation*} \mathbf{E}[ X_{n}]\rightarrow \mathbf{E}[ X] . \end{equation*} These theorems in particular arise when dealing with the convergence of sequences of random variables.

Suppose \(X_{n}\rightarrow X\) almost everywher. Let \(\displaystyle g,h\) be continuous functions with

  1. \(g\geq 0\ \) and \(g( x)\rightarrow \infty \) as \(| x| \rightarrow \infty \),
  2. \(\displaystyle \frac{|h( x) |}{g( x)}\rightarrow 0\) as \(\displaystyle |x|\rightarrow \infty \), and
  3. \(\displaystyle \mathbf{E}[ g( X_{n})] \leq K< \infty \) for all \(\displaystyle n\).

Then \(\mathbf{E}[ h( X_{n})]\rightarrow \mathbf{E}[ h( X)] .\)

A measure space \(\displaystyle ( S,\ \mathfrak{S} ,\ \mu )\) is said to be \(\sigma \)-finite if there is a countable sequence of sets \(\displaystyle A_{i}\) for such that \(\displaystyle \mu ( A_{i}) < \infty \) and \(\displaystyle \bigcup _{i} A_{i} =S\). For example, the Lebesgue measure is \(\displaystyle \sigma \)-finite, since one can take \(\displaystyle A_{i} =( i,\ i+1]\) for all integer \(\displaystyle i\).

Let \(( X,\mathfrak{A} ,\mu _{1})\) and \(( Y,\mathfrak{B} ,\mu _{2})\) be two \(\sigma \)-finite measure spaces. Let \(\displaystyle ( X\times Y,\ \mathfrak{S})\) be defined as follows

\begin{equation*} X\times Y=\{( x,y) :x\in X,y\in Y\} \end{equation*} \begin{equation*} \mathfrak{S} =\{A\times B:A\in \mathfrak{A} ,B\in \mathfrak{B} \} \end{equation*}

Sets in \(\mathfrak{S}\) are called rectangles. It is easy to see that \(\mathfrak{S}\) is a semi-algebra (i.e. closed under intersection, and where the complement can be written as a disjoint finite union of sets in \(\displaystyle S\)): \begin{equation*} ( A\times B) \cap ( C\times D) =( A\times C) \cap ( B\times D) \end{equation*} \begin{equation*} ( A\times B)^{c} =\left( A^{c} \times B\right) \cup \left( A\times B^{c}\right) \cup \left( A^{c} \times B^{c}\right) \end{equation*} Let \(\displaystyle \mathfrak{F} =\mathfrak{A} \times \mathfrak{B}\) be the \(\displaystyle \sigma \)-algebra generated by \(\displaystyle \mathfrak{S}\)

Fubini's theorem. If \(f\geq 0\) or \(\int | f| \ d\mu < \infty \), then \begin{equation*} \int _{X}\int _{Y} f( x,y) \ \mu _{2}( dy) \ \mu _{1}( dx) =\int _{X\times Y} \ f\ d\mu =\int _{Y}\int _{X} f( x,y) \ \mu _{1}( dx) \ \mu _{2}( dy) \end{equation*} In other words, an integral over a rectangle may be computed as the iterated integral over one variable, then the next. This theorem is fundamental to multivariable calculus.

Let \( f( x) =\left(\frac{3}{2}\right)^{x} 2^{-\lfloor x\log_{2}( 3/2)\rfloor }\).

Then:

  • \( a_0 = 1 \)
  • \( a_n \) is the smallest natural number such that \( f(a_n) < f(a_{n-1}) \)

Some values:

n 0 1 2 3 4 5
\(a_n\) 1 2 7 12 53 359

These correspond to \(a_n\)-ET (equal temperament) tunings that are each successively closer to a Pythagorean tuning.
